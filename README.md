💻 Projeto simples em Python — Análise de Logs

Fiz um script em Python para analisar logs do Windows, focando nos eventos de login.
Usei IA como apoio para desenvolver o código, mas aproveitei o processo para entender melhor como funciona leitura de arquivos, uso de regex e organização de dados.
O script basicamente lê o log e ajuda a visualizar padrões de login, como horários com mais acessos.

🛠️ Tecnologias: Python e regex
🔒 Dados anonimizados

É um projeto simples, mas faz parte do meu aprendizado em programação e interesse na área de segurança da informação.
Sugestões e feedbacks são bem-vindos.
